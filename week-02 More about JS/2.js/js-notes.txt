# 🚀 JavaScript Fundamentals: Essential Notes

> A quick-reference guide covering the foundations, core mechanics, and modern ES6+ features of JavaScript.

---

## 🌟 1. Introduction to JavaScript
JavaScript (JS) is a high-level, interpreted, and dynamically typed programming language. It is a core technology of the web.

- **Role:** Adds interactivity, logic, and dynamic behavior to web pages.
- **Execution:** Runs in the browser (V8, SpiderMonkey) and on servers (`Node.js`).
- **Paradigm:** Supports object-oriented, imperative, and declarative (functional) programming styles.

---

## 📦 2. Variables and Declarations
Modern JavaScript uses block-scoped variables. Avoid `var` in modern development.

- **`const`**: 🟢 **Default choice.** Used for values that will *not* be reassigned.
- **`let`**: 🟡 Used for values that *will* change or be reassigned.
- **`var`**: 🔴 *Legacy.* Function-scoped and prone to hoisting bugs.

```javascript
const birthYear = 1995; // ❌ Cannot be reassigned
let currentAge = 25;    // ✅ Can be reassigned
currentAge = 26;        

```

---

## 🏷️ 3. Data Types

JavaScript is **dynamically typed**, meaning variables aren't tied to a specific data type.

### 🧱 Primitive Types (Passed by Value)

* **String:** Text data (`"Hello"`, `'World'`, or backticks ``Hi``)
* **Number:** Integers and floats (`42`, `3.14`)
* **Boolean:** Logical entities (`true` or `false`)
* **Undefined:** Declared but not assigned a value
* **Null:** Intentional absence of any value
* **Symbol / BigInt:** Unique identifiers and massive integers.

### 🔗 Reference Types (Passed by Reference)

* **Objects:** Collections of key-value pairs.
* **Arrays:** Ordered lists of values.
* **Functions:** Reusable blocks of code.

---

## ⚙️ 4. Functions

Functions are **first-class citizens**, meaning they can be stored in variables, passed as arguments, and returned from other functions.

### 📜 Function Declaration (Hoisted)

```javascript
function greet(name) {
  return "Hello, " + name;
}

```

### 🏹 Arrow Functions (ES6)

Shorter syntax, no binding of `this`. Excellent for callbacks.

```javascript
const addNumbers = (a, b) => {
  return a + b;
};

// Implicit return for single-line
const square = x => x * x;

```

---

## 🗂️ 5. Objects and Arrays

### 🧩 Objects

Store collections of related data using key-value pairs.

```javascript
const user = {
  firstName: "Alice",
  role: "Developer",
  greet: function() {
    console.log("Hi, I am " + this.firstName);
  }
};

console.log(user.firstName); // Dot notation
console.log(user["role"]);   // Bracket notation

```

### 📚 Arrays & High-Order Methods

* `map()`: Creates a new array by transforming every element.
* `filter()`: Creates a new array with elements that pass a condition.
* `reduce()`: Accumulates array values into a single output.

```javascript
const numbers = [1, 2, 3, 4];
const doubled = numbers.map(num => num * 2);     // [2, 4, 6, 8]
const filtered = numbers.filter(num => num > 2); // [3, 4]

```

## 🌐 6. The DOM (Document Object Model)

JavaScript interacts with the DOM to change text, styles, or structure dynamically.

**Targeting and Modifying Elements:**

```javascript
// Select elements
const button = document.querySelector('.my-button'); 

// Listen for events
button.addEventListener('click', () => {
  document.body.style.backgroundColor = '#1a1a1a';
});

```

## ⏳ 7. Asynchronous JavaScript

JS is single-threaded. Async JS allows it to initiate tasks (like API calls) without blocking the rest of the code.

### 🤝 Promises

```javascript
fetch('[https://api.example.com/data](https://api.example.com/data)')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error("Error:", error));

```

### ⚡ Async / Await (Modern Approach)

Syntactic sugar over Promises. Makes async code read like synchronous code.

```javascript
const getUserData = async () => {
  try {
    const response = await fetch('[https://api.example.com/user](https://api.example.com/user)');
    const data = await response.json();
    console.log(data);
  } catch (error) {
    console.error("Something went wrong!", error);
  }
};

```

## ✨ 8. Modern ES6+ Magic Features

### 🔤 Template Literals

Inject variables directly into strings using backticks and `${}`.

```javascript
const name = "Bob";
const greeting = `Welcome back, ${name}!`;

```

### ✂️ Destructuring

Extract properties from objects or arrays instantly.

```javascript
const { firstName, role } = user;

```

### 🎇 Spread Operator (`...`)

Expand iterables into individual elements. Perfect for cloning or merging.

```javascript
const arr1 = [1, 2];
const arr2 = [...arr1, 3, 4]; // Result: [1, 2, 3, 4]

```
